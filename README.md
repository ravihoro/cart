# Cart

An online shopping application built using Flutter

Link to apk: https://drive.google.com/file/d/1sz9vpTbRH8thvyFrWdhUzWTmkSmVvfV8/view

## Features
- Google Sign In
- View products
- Add products to favorites/cart
- Dark/Light Theme
- Provider package for state management

## Tools Used
- Flutter
- Firebase
